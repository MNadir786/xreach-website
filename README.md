# XReach LLC Website

A simple, responsive one-page business website for **XReach LLC** offering digital and tax services.

## Tech Stack
- HTML5, CSS3, Vanilla JS
- No build tools or dependencies
- Mobile-first, SEO-ready, accessible

## Structure
```
index.html
style.css
script.js
images/
```

## Deployment
You can host it anywhere:
- **GoDaddy:** Upload all files via File Manager to the website root.
- **Netlify/Vercel:** Drag & drop the project folder to deploy instantly.
- **GitHub Pages:** Push to a repo and enable GitHub Pages in settings.

## Future Enhancements
- Add CMS or backend later for dynamic content
- Integrate a form handler (Node, AWS Lambda)
- Add blog/news section for SEO
- Move to React/Next.js if scaling needed
